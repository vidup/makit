# Instructions pour Claude

## Publication NPM

**Ne JAMAIS exécuter `pnpm publish` ou `npm publish`.**

La publication est faite manuellement par l'utilisateur. Tu peux :
- Mettre à jour la version dans `package.json`
- Préparer les fichiers
- Rappeler à l'utilisateur de publier

Mais c'est l'utilisateur qui exécute la commande de publication.
